## Disable Windows Commands through WSL Distro

1. Create the following file if not already created:
   ```bash
   sudo nano /etc/wsl.conf
   ```

2. Add the following to `wsl.conf`:
    ```ini
    [interop]
    enabled = false
    appendWindowsPath = false
    ```

3. Restart WSL Distro
    ```powershell
    wsl --shutdown
    ```
